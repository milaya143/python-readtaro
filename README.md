# 🌙 Tarot Bot — Инструкция по запуску

## Что нужно
- Сервер или VPS (можно за $5/мес на DigitalOcean / Hetzner / Timeweb)
- Python 3.10+
- Токен от @BotFather (уже вписан в код)

---

## Быстрый запуск

### 1. Загрузи файлы на сервер
```bash
scp tarot_bot.py requirements.txt user@твой-сервер:~/tarot/
```

### 2. Подключись к серверу и установи зависимости
```bash
cd ~/tarot
pip3 install -r requirements.txt
```

### 3. Запусти бота
```bash
python3 tarot_bot.py
```

### 4. Чтобы бот работал постоянно (даже после закрытия терминала)
```bash
nohup python3 tarot_bot.py &
```

---

## Бесплатный хостинг (без сервера)

### Вариант — Railway.app (бесплатно до 500 часов/мес)
1. Зайди на railway.app
2. New Project → Deploy from GitHub
3. Загрузи оба файла в репозиторий
4. Railway сам запустит бота

### Вариант — Render.com (бесплатно)
1. render.com → New → Background Worker
2. Загрузи файлы, укажи команду: `python tarot_bot.py`

---

## Как бот работает

Пользователь пишет /start → видит меню услуг → выбирает → вводит имя →
описывает вопрос → оставляет контакт → подтверждает заявку

Ты получаешь уведомление в Telegram (@ontobe) с полными данными заявки.

---

## Что изменить под себя

В файле tarot_bot.py:
- `ADMIN = "@ontobe"` — замени на свой username если изменится
- `SERVICES = {...}` — можно менять названия, цены, описания
- Тексты сообщений — ищи по русским строкам и правь как хочешь

---

## Команды бота
- /start — начало, показывает меню
- /menu — вернуться к выбору услуги
- /cancel — отменить текущий диалог
